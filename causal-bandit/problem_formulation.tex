Let $q \in [0,1]^N$ be a fixed vector where $q_i = P(X_i = 1)$. In each time-step $t$ upto a known end point $T$:
 
\begin{enumerate}
\item The learner chooses an $I_t \in \set{0,\ldots, N}$ and $J_t \in \set{0,1}$, setting $X_{I_t,t} = J_t$. Selecting $I_t = 0$ corresponds to taking the nothing action $do()$ and just observing. 
\item For $i \neq I_t$, $X_{i,t} \sim Bernoulli(q_i)$

\item The learner observes $ X_t = [X_{1,t} ... X_{N,t}] $
\item The learner receives reward $Y_t \sim \Ber(r(X_t))$, where $r:\set{0,1}^N \to [0,1]$ is unknown and arbitrary.
\end{enumerate}

\vspace{5mm}

The causal structure gives us:
\eq {
P(Y|do(X_i = j)) = & P(Y|X_i = j) \\
= & P(Y|do(X_a = 1),X_i = j)q_a + P(Y|do(X_a = 0),X_i=j)(1-q_a)
}

At each timestep, observing will reveal the reward for half the arms.
